# `audio nonna/` — source recordings + recorded consent

Source recordings of the grandmother (Neapolitan) used to build the demo, plus the
**recorded consent**.

⚠️ **`audio_consenso.ogg` is the recorded consent — keep it.** It has legal value
(authorization to use the voice recordings). Do not delete it or move it out of a safe
location.

The tracks actually played by the HUD live in `web/public/audio/` (named by transcript
turn, matching `spec/fixtures/transcript.jsonl`). This folder holds the **sources** and
the consent, not the demo-wired copies.
